# CFEM-Cloud-File-Encryption-Manager
Encryption is performed locally using industry-standard symmetric encryption (AES) via the .NET cryptography APIs. A user-supplied password is used to derive the encryption key, and encryption occurs before any files are synced to cloud providers such as OneDrive, Dropbox, or Google Drive.
